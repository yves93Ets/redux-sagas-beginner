# redux-saga-beginner

Inspired from [https://redux-saga.js.org/docs/introduction/BeginnerTutorial/]

# Instructions

Setup

```
// clone the repo
git clone https://github.com/yves93Ets/redux-sagas-beginner.git

cd redux-saga-beginner

npm install
```

Run the demo

```
npm start
```

Run tests

```
npm test
```
